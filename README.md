# 📰 News Aggregator

A modern **News Aggregator Web Application** built using **React, TypeScript, Tailwind CSS, and NewsData.io API**.

---

## 🚀 Features

- 📰 Latest News
- 🔍 Search News
- 📂 Category Filter
- 🌙 Dark / Light Mode
- 📅 Publish Date
- 🏢 News Source
- 🔄 Refresh News
- 🌐 Live NewsData API
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- NewsData.io API
- Git & GitHub

## ⚙️ Installation

```bash
git clone https://github.com/AnkitKSaini/news-aggregator.git

cd news-aggregator

npm install

npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file.

```env
VITE_NEWSDATA_API_KEY=YOUR_API_KEY
```
